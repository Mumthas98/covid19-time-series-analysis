# COVID-19 Time Series Analysis using R

## Project Overview
This project focuses on analyzing COVID-19 data using R programming and time series analysis techniques. The analysis includes data cleaning, visualization, trend analysis, and moving averages to understand the spread and impact of COVID-19 over time.

The project was created as part of my learning journey in Data Analytics and Data Science.

---

## Objectives
- Analyze COVID-19 trends over time
- Visualize confirmed, recovered, and death cases
- Perform exploratory data analysis (EDA)
- Apply basic time series analysis techniques
- Create meaningful visualizations using R

---
